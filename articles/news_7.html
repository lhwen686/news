3 月 3 日，Anthropic 发布公告，宣布在其编程工具 Claude Code 中逐步推送全新的「语音模式」，意味着广受开发者欢迎的「直觉编程」正在向更高效的「语音编程」时代演进。

该语音模块直接内置于 Claude Code 的底层架构中，用户在顺利获得内测权限后，系统会在欢迎界面醒目位置显示开启提示。在实际操作环节中，开发者只需长按键盘上的空格键便能直接说话并录入开发需求，随后松开按键即可将语音指令迅速发送给 AI 执行。

费用方面，Anthropic 明确强调，语音交互功能并不会向用户收取任何额外的订阅费用。随着产品推广的逐步深入，现有的 Pro、Max、Team 以及针对大型机构的 Enterprise 层级用户，都将免费体验到这一提升开发效率的全新特性。来源
