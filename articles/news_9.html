派早报：腾讯 QClaw 正式上线、小鹏 P7 发布 Max 版等

💡 AI 深度解析 (AI Insight)

小米MiMo-V2-Pro的发布标志着AI大模型在智能体（Agent）领域的关键进展。智能体是基于大型语言模型构建的自主实体，能够理解复杂指令、进行长程规划并调用外部工具，完成端到端任务。MiMo-V2-Pro的核心技术亮点在于其“混合注意力架构”和对1M超长上下文的支持。混合注意力机制通过优化注意力模式结合，使模型能在拥有万亿级总参数的同时，保持较低的激活参数（42B），从而在扩大能力边界的同时有效控制推理成本和效率。1M超长上下文能力则赋予智能体更强的“记忆力”和情境理解力，使其能处理更复杂的任务流、进行更精细的规划和工具调用，这在处理复杂工作流时不可或缺。

在行业影响方面，MiMo-V2-Pro以媲美顶尖竞品（如Claude Opus 4.6）的使用体验和显著降低的API定价（约五分之一），极大地降低了开发者和企业构建、部署智能体的门槛。这种高性价比的结合有望加速AI Agent技术的普及和商业化落地，推动各行各业的自动化和智能化转型。小米在这一领域的发力，也预示着AI Agent将更深度地融入消费级产品和服务中，带来更智能的用户体验。
Original Text
3 月 19 日，小米发布面向 Agent 场景的旗舰基座模型 Xiaomi MiMo-V2-Pro。官方称，该模型总参数量超过 1T，激活参数为 42B，采用混合注意力架构，支持 1M 超长上下文，重点面向高强度智能体工作流、长程规划和复杂工具调用等现实任务场景。

据小米介绍，MiMo-V2-Pro 在全球模型榜单 Artificial Analysis 上排名全球第八、国内第二，并已在 OpenClaw、Claude Code 等智能体框架中展现端到端任务完成能力，可在无人干预条件下完成复杂工作流编排与精准工具调用。官方称，其整体使用体验已超过 Claude Sonnet 4.6，接近 Opus 4.6，而 API 定价仅为后者约五分之一。

能力层面，MiMo-V2-Pro 延续前代 Hybrid Attention 机制，并将混合比例从 5:1 提升至 7:1，在扩大参数规模的同时维持推理效率；同时通过后训练阶段在更广泛 Agent 任务上的算力扩展，使模型能力从对话和代码生成进一步延伸到完成任务。小米还披露，此前以 Hunter Alpha 名义上线 OpenRouter 的匿名模型，正是 MiMo-V2-Pro 的早期内部测试版本。

目前，MiMo-V2-Pro 已正式开放 API，256K 上下文以内输入价格为每百万 tokens 1 美元、输出 3 美元，1M 上下文以内输入 2 美元、输出 6 美元。该模型也已同步接入 MiMo Studio、WPS 灵犀、小米浏览器及手机端智能体产品 miclaw。来源
