Original Text
这个叫做 Claude Buddy 的彩蛋本来是 Anthropic 给愚人节准备的惊喜，可惜还差临门一脚的时候，Claude Code 的源代码泄漏了，于是全世界都提前知道了 Claude Code 内置了一个「拓麻歌子」。

不过偷跑归偷跑，Claude Buddy 在设计上还是非常可圈可点的。付费订阅用户在最新版 Claude Code 里运行 /buddy ，就可以随机「孵化」出一个以 ASCII 字符画形态呈现的「宠物」。

我抽到的大鹅 Husking

此后，这个宠物会显示在你的 Claude Code 界面上，并且不时跑出来插科打诨一番。Buddy 们说的话也是通过 Claude 生成的——意料之中——并且不计入用量。你也可以主动通过 /buddy pet 命令去摸一摸宠物，而它会回报以满头爱心和一句彩虹屁。

谢谢，有被 mean 到。

根据泄漏的代码可知，Anthoropic 一共定义了 18 个宠物的「物种」，其中既包括鹅、猫、兔、仙人掌、蘑菇这些日常动植物，也有幽灵、机器人、史莱姆这些更古怪的形象。每个宠物都有自己的眼睛造型（共五种）、帽子（共八种）、随机分布的五项「能力值」（纠错能力、耐心、混乱程度、智慧和尖刻程度），以及从「普通」（60% 几率）到「传奇」（1% 几率）共五个等级的稀有度——看来 Anthropic 的工程师是没少玩氪金游戏。

通过代码得知的各种可能组合 (credit: Pickle-Pixel)

Buddy 系统还有严格的「防作弊」机制。每个宠物的形象和属性都是通过用户唯一的 userID 和 Claude Code 代码里写死的一个 salt 值，通过哈希和多次伪随机数运算得出的，理论上共有 37^15（约等于 1.5*10^23）种可能性。因为这些都是纯函数运算，所以同一个账号在任何设备、任何时间打开 Claude Code，永远都会得到同一只宠物。虽然宠物信息会缓存在 ~/.claude 中，但 Claude Code 总是会在启动 Buddy 时重新运算，并且覆盖任何用户自行修改的值。

通过逆向算法得出的所有 buddy 浏览器 (credit: perplexes)

所以，一旦孵化了你的宠物，你唯一的选择就是和它长相厮守了。不过，还是得益于 Claude Code 的代码泄漏，已经有很多好事者做出了 buddy 浏览器，你可以浏览所有可能的形态，甚至通过给 Claude Code 打 patch 来绕过防作弊机制。真是有氪金的地方就有金手指啊。
